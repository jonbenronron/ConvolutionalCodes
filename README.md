# F2Polynomials

## Description

Python file that adds polynomials over field F_2 to numpy.polynomials library.

## Documentation

### Class F2Polynomial

  Coefficients will be given in list or tuple format as an parameter.
  
  #### Parameters:
  coef
  
  #### Methods:
  
  _Initialization_:
  `__init__(self, coef)`
  
  _String format_:
  `__str__(self)`
  
  _Degree of the polynom_:
  `degree(self)`
  
  #### Example:
  
  `p = F2Polynomial((1,2,3))
  print(str(p))`
  
  will print out:
  
  `1 + D^2`
